Case study · Lumi Galaxy

A character universe shaped into an AI-native product loop.

Lumi Galaxy is a playful web world for pixel-alien desktop pets, combining character design, lore structure, and AI-assisted production.

Open live project Back to portfolio
Overview Project phases Product architecture Execution notes Outcome

Product lens

The project explores how a tiny fictional universe can become a product system: characters, knowledge, visual identity, interaction loops, and a web presence that makes the world feel tangible.

Project phases

01

World and character definition

Create the personality system, lore rules, visual cues, and interaction promise for the pixel-alien universe.

02

Knowledge architecture

Structure character facts and world details so they can support future RAG dialogue and consistent behavior.

03

Visual product prototype

Turn the concept into a web experience with motion, planet metaphors, and a playful but controlled art direction.

04

Launch and iteration

Package the project as a shipped indie experiment and define the next loop: dialogue, retention, and richer character interaction.

Product architecture

The case combines a visual layer, a character layer, and a knowledge layer. That structure shows how the project can evolve from a portfolio website into a richer AI companion product with dialogue and persistence.

Execution notes

The build is framed as a solo 0-to-1 workflow. AI tools support ideation, content structuring, web implementation, and iteration, while the product direction keeps a coherent visual language across the planet, characters, and lore.

Outcome

A distinctive creative product case that demonstrates AI-assisted making, RAG-oriented content thinking, and the ability to turn a playful idea into a polished web artifact.
