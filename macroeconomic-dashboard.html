Product lens

Macroeconomic data is abundant but fragmented across providers, frequencies, units, and interfaces. The product creates one exploration layer where users can move from a global signal to a country comparison or historical series without losing context.

Challenge

The hard part was not drawing more charts. It was designing a consistent mental model across GDP, inflation, labour, monetary policy, trade, markets, and fiscal sustainability while preserving the differences that make each indicator meaningful.

Users needed to switch between world, country, and multi-country views without rebuilding the analysis each time.

Annual, quarterly, monthly, and daily series needed shared controls without creating misleading comparisons.

The interface needed to explain signals, not merely display raw numbers.

Building the intelligence layer

01

Define the economic taxonomy

Organize the workspace into six readable categories and define the right unit, frequency, scope, and explanatory text for each indicator.

02

Unify data and interaction rules

Normalize country identifiers, periods, currencies, sources, and chart behavior so each view feels related without hiding data limitations.

03

Design comparison workflows

Connect map selection, rankings, time series, trade categories, currency controls, and multi-country comparison into one synchronized workspace.

04

Add interpretation and delivery

Generate automatic insights from visible data, localize the experience, automate data refreshes, and package the frontend for GitHub Pages.

Co-creation and iteration

This was a co-created project shaped through continuous product and technical iteration. The work required aligning economic meaning, interaction behavior, data constraints, multilingual presentation, and implementation decisions across a shared repository.

Product work focused on making dense macroeconomic information navigable, comparable, and understandable across different user questions.

Technical work established a scalable path from a rich static dataset toward official connectors, APIs, temporal storage, forecasting, and production infrastructure.

Outcome

A live multilingual macroeconomic intelligence product with global scope, deep temporal exploration, synchronized country comparisons, market and trade modules, automatic insights, and a credible path toward a production data platform.
